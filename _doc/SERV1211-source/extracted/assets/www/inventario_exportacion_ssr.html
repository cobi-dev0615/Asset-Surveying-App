Exportación de registros
Para iniciar la exportación capture los siguientes datos manualmente y presione el botón Iniciar exportación.
* Verifíque que los datos capturados sean correctos para evitar errores.
* Este proceso requiere conexión WiFi en la misma red en la que está conectado su servidor.
* Por favor no cierre la aplicación ni suspenda el equipo para evitar un problema de datos repetidos.
